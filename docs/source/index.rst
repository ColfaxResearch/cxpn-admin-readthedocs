Welcome to Colfax's CXPN Admin documentation!
=============================================

This guide covers the **WP Colfax Connect** WordPress plugin, which powers the Colfax Experience Center test drive portal. The plugin manages the full user lifecycle: from registration and access provisioning, through system seat reservations, to post-session feedback collection. It integrates with external Colfax backend services via JWT-authenticated REST APIs.

Check out the :doc:`get-started` section to get a primer on all of the features and settings available to you as an admin. :doc:`access-management` contains details on how to manage your users, including how to :ref:`inviteusers` or :ref:`removeusers`.

For information on how to customize granular controls of the portal, check out :doc:`portal-settings`.

.. note::

   This project is under active development.
